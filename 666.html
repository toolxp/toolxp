常见问题解答

IP/DNS 泄露会导致哪些风险？

IP 或 DNS 泄露可能会暴露你的真实身份和网络行为，带来以下风险：

  • 暴露真实位置，影响隐私安全；
  • 绕过 VPN 或代理，导致匿名性失效；
  • DNS 请求被第三方监控，泄露你的上网浏览记录；
  • 可能成为黑客攻击目标，如 DDoS 或钓鱼网站劫持。
检测结果显示“IP 未泄露”代表什么？

说明你的浏览器仅暴露了一个 IP，可能的原因包括：

  • 你未使用代理或 VPN；
  • 你使用了代理/VPN，且其配置安全、未发生泄露。
检测结果显示“检测到多个不同的 IP”代表什么？

说明你的浏览器暴露了多个来自不同自治系统（AS）的 IP。这通常意味着你正在使用代理或 VPN，并且存在 IP 泄露的可能，真实 IP 可能已被检测到。

为什么页面上显示“未检测到”？

有以下几种可能：

  • 你目前的网络波动较大，连接不稳定；
  • 你做了一些有效的隐藏措施；
  • 你是单栈网络；
  • 你是双栈网络，默认 IPv4/v6 优先；
  • 你的 WebRTC 已关闭或隐藏了 IP；
  • 检测通道不稳定。
如何判断我的 IP/DNS 是否泄露？

此工具会使用多个检测源检测公网 IP 地址。如果你在已经开启了 代理/VPN 的情况下, 真实 IP 和本地的 DNS 仍然被检测到，可以确认已经泄露 。

如何查询我的 IP 纯净度、综合质量？

你可以使用 IP质量检测工具 进行全面检测。

为什么我的 IP 地址会变化？

如果你的 ISP 提供的是 **动态 IP**，那么你的 IP 地址可能会在每次重新连接网络时发生变化，更换 VPN 和 代理，Wi-Fi ，移动数据，也会发生变化。

如何隐藏我的 IP 地址，防止 IP 地址被追踪？

要防止 IP 地址被追踪，你可以：

  • 使用可靠的 VPN 或代理服务器隐藏真实 IP；
  • 关闭分流代理并开启 TUN 模式；
  • 定期更换网络或动态 IP 地址；
  • 禁用 WebRTC 以防止真实 IP 泄露；
  • 使用匿名浏览器或 Tor 网络增强隐私保护；
  • 避免在社交媒体暴露 IP 相关信息。

这些措施可以有效减少第三方对你真实 IP 地址的跟踪，提高在线隐私安全性。
